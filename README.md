# 목재 재단 로스율 자동 계산기 (Streamlit)

## 추가 반영된 기능
- 규격상세 파싱 확장: `*`, `x`, `X`, `×` 지원
- **실원장수(장)** 입력 시 실제 원장 사용수 기준으로 사용면적/로스 재계산
- **자투리 이월(재고)**:
  - 입력: 기초 자투리 재고(m²), 생산일별 자투리입고(m²), 자투리사용(m²)
  - 계산: 기말 = 기초 + 입고 - 사용 (재고 음수 불가, 초과 사용은 자동 제한)
- 분석 탭: 제품코드/색상/부품명/두께별 조정로스율(%) + 재단점유율(%)

## 실행
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Streamlit Cloud 배포
- Main file path: `app.py`
