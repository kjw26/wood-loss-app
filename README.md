# 수율(로스율) 프로그램 · 생산일별 표 + 합계 (v2)

## 반영 내용
- 1페이지에서 **생산일별로 컬럼 표시**하고, 마지막에 **합계** 컬럼 추가
- 자투리 보관 사용을 입력하면 기본적으로 **신규 자재투입을 대체**한다고 가정하여
  - 자재투입면적 = (원장수×유효원장면적) - 자투리사용면적
  - 따라서 자투리 사용이 늘면 loss율이 **감소**하는 방향
- 필요 시 사이드바에서 “투입면적에 더함(참고용)”으로 변경 가능
