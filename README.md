# 목재 재단 로스율 자동 계산기 (Streamlit)

## 기능
- 엑셀 업로드
- **① 요약 탭**: 생산일별 사용면적/로스면적/로스율(%) + 전체 합계
  - 생산일별 `자투리사용(m²)` 입력 시 조정 로스율 자동 감소
- **② 분석 탭**: 제품코드/색상/부품명/두께별 로스율 + 재단점유율(순면적 비중)
- 옵션값(원장/테두리여유/날물두께) **사이드바에서 보정 가능**
- 결과 엑셀 다운로드(설정/요약/분석/Data 포함)

## 필수 컬럼(기본)
- 생산일
- 규격상세 (예: 300*500*18)
- 생산량
- 제품코드
- 색상
- 부품명

> 컬럼명이 다르면 app.py의 `pick_col()` 후보 리스트에 추가하면 됩니다.

## 실행 방법
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Streamlit Cloud 배포
- Main file path: `app.py`

## 규격상세 파싱 추가 지원(업데이트)
- 규격상세에 텍스트가 앞/뒤에 붙어 있어도 인식합니다. 예: `(ra 2ea) 808*1100*22(PB)`
