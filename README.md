# 목재 재단 로스율 계산 & 부품명 분석 (색상 구분 v3)

## 핵심 변경
- 색상 컬럼명은 **'색상'으로 고정**
- 색상별 로스율(면적 하한 + 네스팅 추정) 계산
- 부품명×색상: "해당 색상의 로스율(%)" 및 "로스 기여(추정)" 제공
  - 로스 기여(추정) = 색상별 Waste 면적을 부품 면적비중으로 분배한 값

## 화면 구성
- 요약: 전체 로스율
- 색상별 로스율: 색상별 LossMin_% / LossNest_%
- 부품명×색상(로스 기여): 색상별 로스율과 부품별 기여도
- 부품명/규격 분석: 전체 ABC
- 오류/예외: 파싱 실패, 치수초과
